📚 Introdução

O Algoritmo de Euclides é um dos algoritmos mais antigos e elegantes da matemática, desenvolvido pelo matemático grego Euclides por volta de 300 a.C.

🌟 Aplicações Práticas Modernas

  • 🔐 Criptografia RSA: Fundamental para gerar chaves criptográficas
  • 📊 Simplificação de frações: Reduzir frações à forma irredutível
  • 🎨 Computação gráfica: Geração de padrões e tessellações
  • ⚙️ Teoria dos números: Base para algoritmos avançados
Definição do Problema: Dados dois números inteiros positivos a e b, encontrar o maior número inteiro que divide ambos sem deixar resto - o MDC(a,b).

💻 Implementação Recursiva

// Versão Recursiva do Algoritmo de Euclides
function mdcRecursivo(a, b) {
    // Caso base: quando b = 0, o MDC é a
    if (b === 0) {
        return a;
    }
    // Chamada recursiva: MDC(a, b) = MDC(b, a % b)
    return mdcRecursivo(b, a % b);
}

// Exemplo de uso
console.log(mdcRecursivo(48, 18)); // Output: 6
console.log(mdcRecursivo(100, 25)); // Output: 25

🔄 Implementação Iterativa

// Versão Iterativa (otimizada para performance)
function mdcIterativo(a, b) {
    // Garante que trabalhamos com valores absolutos
    a = Math.abs(a);
    b = Math.abs(b);

    // Loop enquanto b não for zero
    while (b !== 0) {
        let temp = b;
        b = a % b;  // Novo b é o resto da divisão
        a = temp;   // Novo a é o antigo b
    }
    return a;
}

🔄 Implementação Iterativa com rastreamento para análise

// Versão Iterativa com rastreamento
function mdcComPassos(a, b) {
    let passos = [];
    let stepCount = 0;

    while (b !== 0) {
        let quociente = Math.floor(a / b);
        let resto = a % b;
        passos.push(`${a} = ${b} × ${quociente} + ${resto}`);

        a = b;
        b = resto;
        stepCount++;
    }
    return { resultado: a, passos: passos, iteracoes: stepCount };
}

📐 Análise Matemática de Complexidade

Relação de Recorrência

T(a, b) = T(b, a mod b) + O(1)

🏆 Teorema de Lamé (1844)

O número de passos é no máximo 5 vezes o número de dígitos decimais de b.

Número de passos ≤ logφ(√5 × b)
onde φ = (1 + √5)/2 ≈ 1.618 (razão áurea)

✅ Melhor Caso

O(1) - quando b divide a perfeitamente

⚠️ Pior Caso

O(log n) - números de Fibonacci consecutivos
